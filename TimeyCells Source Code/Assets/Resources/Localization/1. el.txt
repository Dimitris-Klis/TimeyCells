# Εισαγωγή:
Το _TimeyCells_ είναι μια νέα έκδωση της προηγούμενής μου εφαρμογής, _School Timetable_. Η προηγούμενη εφαρμογή _School Timetable_ δεν είχε πολλή προσαρμοστικότητα και δημιουργήθηκε για να χρησιμοποιηθεί μόνο από Κύπριους μαθητές. Αντίθετα, η _TimeyCells_ χτίστηκε από την αρχή με έναν στόχο: Η οργάνωση να γίνει όσο το δυνατόν περισσότερο εύκολη και αποτελεσματική.<br/><br/>
Μπορείτε να δημιουργήσετε συμβάν, είδη συμβάν, κελιά πολλαπλών γραμμών και μπορείτε να ανταλλάξετε γραμμές και στήλες. Μπορείτε ακόμη να αλλάξετε τα χρώματα της εφαρμογής σε μία από τις προκαθωρισμένες επιλογές ή να δημιουργήσετε το δικό σας χρωματοθέμα! Επιπλέον, μπορείτε να αλλάξετε τη διάρκεια ή την ώρα έναρξης ενός συγκεκριμένου μαθήματος ή συμβάν. Τέλος, μπορείτε να αντικαταστήσετε πρωσωρινά τα συμβάν και θα ενημερώνεστε για το πιο συμβάν συμβαίνει εκείνη την στιγμή και σε πόση ώρα τελειώνει.
<br/>
# Συμβάν, Είδη Συμβάν:
Για να δημιουργήσετε συμβάν και είδη συμβάν, πρέπει να πάτε στην καρτέλα 'Επεξεργασία Συμβάν':
!GIF[BurgerEvents]
<br/>
## Δημιουργία Ειδών Συμβάν:
Τα είδη συμβάν χρησιμοποιούνται για την κατηγοριοποίηση των συμβάν της εβδομάδας. Μπορείτε να αλλάξετε τα χρώματα του κειμένου και του φόντου. Τα είδη συμβάν επίσης χρειάζονται ένα όνομα.
!GIF[EventTypeCreate]
<br/>
## Επεξεργασία Ειδών Συμβάν:
Για να επξεργαστείτε ένα είδος συμβάν, απλά πατήστε πάνω σ' αυτό που θέλετε να αλλάξετε. Μπορείτε επίσης να αλλάξετε τα χρώματα του default είδους συμβάν.
!GIF[EventTypeEdit]
<br/>
## Δημιουργία Συμβάν:
Τα συμβάν έχουν 5 ιδιότητες:
- <indent=5%>**Όνομα:** Το όνομα του συμβάν.</indent>
- <indent=5%>**Λεπτομέριες 1:** Χρήσιμο για επιπρόσθετες λεπτομέριες, όπως η τοποθεσία του συμβάν, ή το όνομα του οργανωτή του συμβάν.</indent>
- <indent=5%>**Λεπτομέριες 2:** Χρησιμοποιείται επίσης για επιπρόσθετες λεπτομέριες.</indent>
- <indent=5%>**Είδος Συμβάν:** Χρήσιμο για την κατηγοριοποίηση του συμβάν, το οποίο σας βοηθάει να καταλάβετε πόσο σημαντικό είναι το μάθημα/συμβάν.</indent>
- <indent=5%>**Αγαπημένο:** Μία απλή ιδιότητα που δείχνει πόσο σας αρέσει ένα συμβάν.</indent>
!GIF[EventCreate]
<br/>
## Επεξεργασία Συμβάν
Για να επξεργαστείτε ένα συμβάν, απλά πατήστε πάνω σ' αυτό που θέλετε να αλλάξετε.
<br/>

# Χειροκίνητη Επεξεργασία<br/>
## Επεξεργασία Ημέρας:
Οι ημέρες έχουν τις εξής ιδιότητες:
- <indent=5%>**Όνομα:** Το όνομα της ημέρας.</indent>
- <indent=5%>**Ώρα Έναρξης:** Η ώρα στην οποία ξεκινά η ημέρα.</indent>
- <indent=5%>**Κοινή Διάρκεια:** Η κοινή διάρκεια για κάθε κελί.</indent>
- <indent=5%>**Ημέρες:** Ποιες μέρες θα συμβαίνει η ημέρα.</indent>
!GIF[WeekdayEdit]
<br/>
## Επεξεργασία Κελιών:
Όταν δεν επεξεργάζεστε τον πίνακα, πατώντας πάνω σ' ένα κελί θα εμφανίστει ο επεξεργαστής κελιού. Εδώ, μπορείτε να επεξεργαστήτε χειροκίνητα το επιλεγμένο συμβάν ή να αντικαταστήσετε μερικές από τις ιδιότητες του συμβάν. Μπορείτε επίσης να αντικαταστήσετε την διάρκεια του κελιού ή/και την ώρα έναρξης. Αν δεν αντικατασταθούν, η διάρκεια θα έιναι ίση με την κοινή διάρκεια της γραμμής/ημέρας.
!GIF[CellEdit]
<br/>
## Επεξεργασία Ετικετών:
Οι ετικέτες είναι τα μικρά κελιά που βρίσκονται πάνω από τον πίνακα. Δείχνουν την ώρα έναρξης και έναν δείκτη για κάθε κελί της σημερινής ημέρας. Μία ετικέτα μπορεί να είναι άδεια αν το κελί είναι άδειο ή η σημερινή ημέρα δεν αντιστοιχεί πουθενά.
<br/>
Μπορείτε να επεξαργαστήτε μία ετικέτα ώστε ο δείκτης της να εμφανίζει πάντα κείμενο.
!GIF[CustomLabels]
<br/>
Μπορείτε επίσης να κάνετε την ετικέτα να μετρήσει και ως δείκτης, που σημαίνει ότι οι υπόλοιποι δείκτες θα εμφανίζονται ως:
`1, 2, 3, 4, ΚΕΙΜΕΝΟ, 6, 7, 8`
Και όχι:
`1, 2, 3, 4, ΚΕΙΜΕΝΟ, 5, 6, 7, 8`<br/>
Να πως θα φαίνονταν οι ετικέτες αυτού του πίνακα σε μια Τρίτη:
!IMG[Tuesdays]
_* Η τελευταία ετικέτα είναι άδεια, εφόσον το τελευταίο κελί της Τρίτης είναι άδειο._
<br/>
## Προσωρινές αλλαγές:
Οι προσωρινές αλλαγές αντικαθιστούν τις ιδιότητες μιας ημέρας ή ενός κελιού με καινούριες για έναν αριθμό εβδομάδων. Μπορείτε επίσης να καθυστερήσετε την αλλαγή για έναν αριθμό εβδομάδων. Αν η διάρκεια της αλλαγής είναι μηδενική, Η αλλαγή θα λήξει την επόμενη φορά που η συγκεκριμένη ημέρα θα συμβεί.
_ΣΗΜΕΙΩΣΗ: Αν μια ημέρα/γραμμή συμβαίνει για πάνω από μία ημέρα, η αλλαγή θα λήξει στην πιο μακρινή ημέρα._

# Αντιγραφή & Επικόλληση Δεδομένων
Αυτό είναι το κουμπί <u>αντιγραφής</u>:
!OBJ[Copy]
<br/>
Και αυτό είναι το κουμπί <u>επικόλλησης</u>:
!OBJ[Paste]
<br/>
Με αυτά τα 2 κουμπιά, επαναλαμβανόμενα σημεία του πίνακα μπορούν να επεξεργαστούν συντομότερα. 
<br/>
Μπορείτε επίσης να χρησιμοποιήσετε αυτά τα δύο κουμπιά για να μοιραστείτε τον πίνακά μας με άλλα άτομα. Όταν γίνεται μια αντιγραφή, τα δεδομένα του πίνακα αντιγράφονται ως κείμενο. Αυτό το κείμενο, μπορεί έπειτα να σταλεί σε οποιονδήποτε.
<br/>
Όταν ο δέκτης λάβει το κείμενο, αν έχει την εφαρμογή, μπορεί να το αντιγράψει και να πατήσει το κουμπί επικόλλησης για να μπορέσει να χρησιμοποιήσει τον πίνακά σας.

# Φορητή Λειτουργία (Μόνο σε Η/Υ)
Για να ενεργοποιήσετε την Φορητή Λειτουργία, πρέπει να επεξεργαστείτε το αρχείο ` ExtraProperties.json `, το οποίο βρίσκεται στο
<size=80%>` ΦΑΚΕΛΟΣ-ΕΦΑΡΜΟΓΗΣ\TimeyCells\_Data\StreamingAssets\ `</size>,
με έναν επεξεργαστή κειμένου και να αντικαταστήσετε το ` "IsPortable": false ` με ` "IsPortable": true `.
<br/>
Όταν η Φορητή Λειτουργία είναι ενεργοποιημένη, όλα τα αρχεία θα αποθηκεύονται στο φάκελο της εφαρμογής αντί για το φάκελο ` %AppData% ` του Η/Υ. Αυτή η λειτουργία είναι χρήσιμη, αν προτιμάτε να έχετε την εφαρμογή μέσα σ' ένα USB η άλλη αποθηκευτική μονάδα που συνδέεται με πολλούς υπολογιστές.
